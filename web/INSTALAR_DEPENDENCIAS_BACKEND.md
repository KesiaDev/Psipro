# 🔧 Instalar Dependências do Backend

## ⚠️ Erro: "'nest' não é reconhecido"

Este erro acontece porque as **dependências do backend não estão instaladas**.

---

## ✅ Solução: Instalar Dependências

### Passo 1: Ir para pasta do Backend

```powershell
cd C:\Users\User\Desktop\psipro\backend
```

### Passo 2: Instalar Dependências

```powershell
npm install
```

**Isso vai demorar alguns minutos** (baixar todas as dependências).

### Passo 3: Após Instalar, Iniciar Backend

```powershell
npm run start:dev
```

---

## 📝 Resumo dos Comandos

```powershell
# 1. Ir para backend
cd C:\Users\User\Desktop\psipro\backend

# 2. Instalar dependências (primeira vez)
npm install

# 3. Iniciar backend
npm run start:dev
```

---

## ⏱️ Tempo Estimado

- `npm install`: 2-5 minutos (dependendo da internet)
- `npm run start:dev`: 10-30 segundos

---

## ✅ Quando Funcionar

Você deve ver:
```
[NestApplication] Nest application successfully started on port 3001
```
